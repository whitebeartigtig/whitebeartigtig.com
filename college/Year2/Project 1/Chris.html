Animation Basics

I created a keyframe at frame 1 [S] and then rotated it a bunch and selected frame 80 and made another keyframe.

Press [D] to edit pivot, then you can move it into place manually using the arrows or click on a face or vertex on the object. Holding [SHIFT] moves the position of the pivot but keeps the orientation. Holding [CTRL] + [SHIFT] aims the pivot to wherever you are selecting.

In animation, to drop a keyframe, press [S]. This will put a red marker on the timeline. We put a marker at the start and end of every movement and we place a marker at the start position of the object, move to the frame where we want the movement to end, move/rotate/scale the object and then press [S] again to save the new position.

Milk Jug Model

After creating a new project, I created a new cube with the dimensions of 1x1x1 with Width, Height and Depth divisions set to 3.

I then selected all of the corners in vertex mode and moved them inwards.

I then selected some faces and extruded them upwards with one division.

I then cut some holes ready for adding in the handle.

After that, I selected the edges of the holes and bridged them together, added 2 divisions and set the Curve Type to "Blend".

Texture Layers

Base Colour is simply the colour of the texture. It is also known as Albedo or Diffuse Map

Metalness is how metallic it is, black being not at all, and white being very metallic

Roughness would be how rough the texture is

Normal map is height?

AO map looks to be brightness

Opacity is pretty self explanitory

Specular F0 looks to be just a blank shape

This is the process of going from just a simple, flat texture to something much nicer using the different maps. First using Base Colour, then Bump Map, then Roughness, then Displacement and then fixing up the shapes by changing the scale and increasing the iterations.

You can also do the same thing using the Substance plugin. It makes it a lot easier.

Animating a wind turbine

I made the terrain and added the texture, along with playing around with the exposure of the textures to get the colour right.

Now we are going to animate the wind turbine, first by moving the pivot to make the wind turbine spin correctly.

Select frame 1, press S to create a keyframe, then select frame 200, and change the rotate z, in this case I did -1080. Now the turbine spins.

To change the easing on the animation, we can go to the graph editor and change it to linear.

I created a camera, and snapped the aim onto the wind turbine.

I adjusted these render settings

A: The name of your render video

B: Image format (PNG is lightweight)

C: Important that Maya is not set to single frame. In this example our files will be named as follows:

frame.1.png

frame.2.png

etc…

D: Set the range you want to render

E: Choose the correct camera

F: Choose the resolution/quality.

To render, I switched to the rendering mode, went to render, and clicked render sequence. This will now render out the entire 200 frames that we have animated. This will take a while.
